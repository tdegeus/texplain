Getting texplain
================

Using conda
-----------

.. code-block:: none

  conda install -c conda-forge texplain

Using PyPi
----------

.. code-block:: none

  python -m pip install texplain

From source
-----------

.. code-block:: none

  git checkout https://github.com/tdegeus/texplain.git
  cd texplain

  python -m pip install .
